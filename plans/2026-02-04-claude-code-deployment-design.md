# Claude Code Windows 11 部署教程设计文档

## 设计日期
2026-02-04

## 目标
为完全没有编程经验的普通用户创建一套完整的 Claude Code 在 Windows 11 环境下的部署教程。

## 目标用户
- 完全没有编程经验的普通用户
- 不熟悉命令行操作
- 需要从零开始学习所有基础概念

## 文档范围
仅涵盖安装和基础配置，包括：
- 命令行基础知识
- Node.js 环境安装
- 环境变量配置
- Claude API Key 获取
- Claude Code 安装
- 基础配置和验证
