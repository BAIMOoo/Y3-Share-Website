# 简介

Y3-Share 是一个围绕 Y3 编辑器实验、AI 辅助开发、服务部署和实用开发笔记的个人知识库。

## 推荐阅读路径

- <strong>新读者</strong>：先从本页开始，再阅读“关于Y3QA”，了解站点和问答入口。
- <strong>AI 辅助 Y3 开发</strong>：阅读 Y3AI 教程，重点看专业名称释义、Agent 部署和 Y3 项目开发流程。
- <strong>服务部署</strong>：阅读 Y3Tcp 或 Y3Websocket 服务部署教程。
- <strong>实用技巧</strong>：阅读面对面会议分享文档。这些页面会刻意保留现场经验、个人判断和零散小技巧。

## 时效说明

AI 工具、第三方服务、下载链接和截图都可能随时间变化。阅读时，涉及时效性的部分请优先参考各文档中的验证说明、官方链接和风险提示。
