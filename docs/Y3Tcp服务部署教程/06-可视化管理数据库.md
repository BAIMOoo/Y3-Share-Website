# 06-可视化管理数据库

:::warning 部署与安全说明
服务端工具、数据库、Docker Compose 和 Nakama 可能随版本变化。示例端口、默认用户、密码和公网 IP 仅适合学习或测试；生产环境请使用强密码、最小化暴露端口，并以当前官方文档为准。
:::
