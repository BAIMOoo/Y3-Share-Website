# 模型&Agent对比

:::tip 验证与时效说明
AI 工具、模型名称、安装步骤和第三方服务变化很快。本文保留实践经验，但涉及时效性的内容，生产使用前请结合官方文档和来源记录再确认。
:::


本文撰写与2026年2月9日，主要对比当下主流、前沿的编程大语言模型：Claude、ChatGPT-Codex、Gemini

<strong>以下数据均为个人感受，无具体量化数据，且测试时Code5.3和Claude Ops4.6都没公布，所以下文仅性价比可作简单参考</strong>



## 响应速度

Claude Code CLI+Claude Ops 4.5  >  opencode+Codex5.2 >  opencode+Gemini 3 pro

## 性价比

Codex > Gemini >> Claude

## 代码生成质量

Claude Code CLI+Claude Ops 4.5 > opencode+Codex5.2 > opencode+Gemini 3 pro
