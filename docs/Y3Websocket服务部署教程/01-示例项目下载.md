# 01-示例项目下载
1. 下载演示项目https://gitee.com/baim00/y3-web-socket-project/repository/archive/master.zip

    本示例项目包含完整的Y3项目，以及一个nakama服务端，可以利用docker快速部署

    项目地址：[BAIM00/Y3WebSocketProject](https://gitee.com/baim00/y3-web-socket-project)

    该Y3项目下载解压即可正常运行，默认连接网络上公开的ws回显服务器，大家可以下载这个项目后直接体验功能（启动游戏后按空格即可触发测试逻辑）。


2. 脚本目录位于项目下 maps\EntryMap\script

    其中入口函数和测试逻辑均写在 maps\EntryMap\script\main.lua

    Lua Websocket脚本目录位于 maps\EntryMap\script\websocket.lua  

    Python脚本目录位于 maps\EntryMap\custom_eca\custom_python.py

    nakama服务端目录位于 maps\EntryMap\script\backend\nakama

    nakama-lua-sdk 脚本目录位于 maps\EntryMap\script\nakama-lua-sdk 【非必须文件， 仅在演示项目中展示如何链接开源游戏服务端框架】

3. 如何让老项目能使用websocket？

    a. 拷贝Python脚本到您的项目关卡下的相同路径（每个关卡下的\custom_eca\custom_python.py）

    b. 编辑器内保存项目（必要步骤，否则lua无法调用到定义的py函数）

    c. 拷贝Lua脚本到您的项目关卡下的相同路径


4. 如何更新客户端代码？
    重新执行1-4步即可
